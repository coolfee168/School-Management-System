# School Management System

A simple desktop app built with C# Windows Forms for managing students, teachers, classes, and payments. Made as a school project.

## What it does

- Add and manage student records
- Add and manage teacher records
- Create and organize classes
- Track student payments
- View a dashboard with an overview of everything

## Built with

- C# (.NET)
- Windows Forms
- Visual Studio

## How to run it

1. Clone this repo
2. Open `School Management System.slnx` in Visual Studio
3. Build the project (Ctrl+Shift+B)
4. Run it (F5)
